# Cardboard Pet

Minecraft用のMOD。召喚者を守り、敵対Mobを狩る段ボール姿のエンティティ「段ボール」を追加します。

このリポジトリには2つのバージョンが含まれています。

- ルート（このディレクトリ） — **Minecraft 1.20.1 / Forge**
- [`neoforge-1.21.1/`](neoforge-1.21.1/) — **Minecraft 1.21.1 / NeoForge**（同じ機能の移植版。詳細は[neoforge-1.21.1/README.md](neoforge-1.21.1/README.md)）

以下はForge 1.20.1版の説明です。

## 動作環境

- Minecraft 1.20.1
- Forge 47.4.10 以降
- Java 17 以上

## 追加要素

### 段ボール召喚器

ネザライトのクワを使ってクラフトする専用アイテム。地面を右クリックすると、その場に「段ボール」を召喚します。召喚したプレイヤーがオーナーとして記録されます。

```
[紙] [紙] [紙]
[紙] [ネザライトのクワ] [紙]
[紙] [糸] [紙]
```

### 段ボール呼び笛

紙と糸でクラフトする専用アイテム。使用すると、自分がオーナーの段ボールを全てプレイヤーの近くにテレポートさせて呼び寄せます。

```
[ ] [紙] [ ]
[紙] [糸] [紙]
[ ] [紙] [ ]
```

### 段ボール巨大化の杖

紙・ブレイズロッド・アメジストの欠片・段ボール召喚器でクラフトする専用アイテム。地面を右クリックすると、その場に通常の2.5倍サイズの「段ボール」を召喚します。体力・攻撃力も同倍率で強化されます。召喚したプレイヤーがオーナーとして記録されます。連続使用防止のクールダウン（5秒）があり、消費アイテムではありません。

```
[ ] [アメジストの欠片] [ ]
[紙] [ブレイズロッド] [紙]
[ ] [段ボール召喚器] [ ]
```

### 段ボールの剣

紙2枚と棒でクラフトする専用アイテム。攻撃力は木の剣相当ですが、攻撃速度は3.6（木の剣は1.6）と非常に軽くて素早い、段ボール製の剣です。段ボールにシフト＋右クリックで持たせて武器にすることもできます。刃は半透明で、暗闇でも奥から発光色が透けて見えます（エミッシブ表示）。

```
[紙]
[紙]
[棒]
```

### 石強化段ボールの剣

段ボールの剣とコブルストーン2個でクラフトする専用アイテム。攻撃力は石の剣相当に強化されますが、攻撃速度は段ボールの剣と同じ3.6を維持します。刃がわずかに灰色がかった見た目になります（エミッシブ発光は段ボールの剣と同じ）。

```
[段ボールの剣] [コブルストーン] [コブルストーン]
```
（クラフト台の好きなマスに配置可能）

### 深層岩強化段ボールの剣

段ボールの剣とコブル深層岩2個でクラフトする専用アイテム。攻撃力は石強化より0.5低いですが、耐久力はかなり高くなります（段ボールの剣からの並列アップグレードで、石強化を経由する必要はありません）。攻撃速度は3.6を維持。刃がわずかに黒みがかった見た目になります。

```
[段ボールの剣] [コブル深層岩] [コブル深層岩]
```
（クラフト台の好きなマスに配置可能）

### 段ボール（Cardboard Box）

召喚したプレイヤーを守り、敵対Mobを狩るMob。誰彼構わず襲いかかることはありません。

- **攻撃対象**
  - 敵対Mob（ゾンビ・スケルトンなど）は無条件で攻撃します
  - オーナーが誰かを殴った・攻撃された場合、その相手を攻撃します（援護・護衛）
  - 段ボール自身が殴られた場合、殴った相手を攻撃します（反撃。ただし他の段ボールへの反撃はしません）
  - 上記に当てはまらない動物やプレイヤーは放置します
- **オーナー不可侵**: 召喚したプレイヤーは絶対に攻撃しません。殴られても反撃しません
- **同族不可侵**: 段ボール同士は互いに攻撃しません
- **飛行突撃**: 対象から離れていると、まれに重力を無視して空中を直進し対象へ突撃します。射程に入ると飛んでいる最中でも攻撃が命中します（着地を待つ必要はありません）。着地時はゆっくり降下（低速落下）に切り替わります
- **永続化**: ワールドを離れてもオーナー情報を保持し、遠距離でもデスポーンしません
- **オーナー死亡時に消滅**: 召喚したプレイヤーが死亡すると、その段ボールは自動的に消えます。代わりに段ボール召喚器が1つ、オーナーの次のリスポーン時に付与されます（`keepInventory`の設定に関わらず受け取れます）
- **装備**: オーナーがシフト＋右クリックすると、手に持っているアイテムを段ボールに着脱できます（アイテムを持って右クリックで装着、空手で右クリックで外して受け取る）。防具（ヘルメット/チェストプレート/レギンス/ブーツ）は持っている手に関わらず対応する防具スロットへ、それ以外のアイテムはメインハンド／オフハンドへ装着されます。オーナー以外は装備を変更できません。防具の見た目は現状反映されません（モデルに頭部パーツが無いため）
- **装備の返却**: 段ボールが倒された場合、装着していたアイテムは地面に落ちるのではなくオーナーへ直接返却されます（オーナーが読み込まれていない場合や、インベントリが満杯の場合は足元にドロップ、それも不可能な場合は段ボールの位置にドロップ）

## ビルド・実行方法

```
git clone <このリポジトリのURL>
cd cardboard-pet
./gradlew runClient
```

初回実行時はMinecraftのソースのダウンロード・デコンパイルが走るため、時間がかかります。

MODのjarをビルドする場合:

```
./gradlew build
```

`build/libs/` にjarファイルが生成されます。
