# Takumi Mod

Minecraft Forge 1.20.1 用のMOD。召喚者以外のすべての生物を自動的に攻撃する、段ボール姿のエンティティ「段ボール」を追加します。

## 動作環境

- Minecraft 1.20.1
- Forge 47.4.10 以降
- Java 17 以上

## 追加要素

### 段ボール召喚器

ネザライトのクワを使ってクラフトする専用アイテム。地面を右クリックすると、その場に「段ボール」を召喚します。召喚したプレイヤーがオーナーとして記録されます。

```
[紙] [紙] [紙]
[紙] [ネザライトのクワ] [紙]
[紙] [糸] [紙]
```

### 段ボール（Cardboard Box）

召喚者を除くすべての生物（他プレイヤー・モンスター・動物）を自動的に索敵し、近接攻撃で襲いかかるMob。

- **オーナー不可侵**: 召喚したプレイヤーは絶対に攻撃しません。殴られても反撃しません
- **同族不可侵**: 段ボール同士は互いに攻撃しません
- **跳躍攻撃**: 対象に近づくと飛びかかって攻撃します
- **永続化**: ワールドを離れてもオーナー情報を保持し、遠距離でもデスポーンしません
- **オーナー死亡時に消滅**: 召喚したプレイヤーが死亡すると、その段ボールは自動的に消えます

## ビルド・実行方法

```
git clone <このリポジトリのURL>
cd takumimod
./gradlew runClient
```

初回実行時はMinecraftのソースのダウンロード・デコンパイルが走るため、時間がかかります。

MODのjarをビルドする場合:

```
./gradlew build
```

`build/libs/` にjarファイルが生成されます。
