# Cardboard Pet (NeoForge 1.21.1)

Minecraft 1.21.1 / NeoForge 用のMOD。[ルートのForge 1.20.1版](../README.md)と同じ機能を持つ移植版です。召喚者を守り、敵対Mobを狩る段ボール姿のエンティティ「段ボール」を追加します。

## 動作環境

- Minecraft 1.21.1
- NeoForge 21.1.235 以降
- Java 21 以上

## 追加要素

### 段ボール召喚器

ネザライトのクワを使ってクラフトする専用アイテム。地面を右クリックすると、その場に「段ボール」を召喚します。召喚したプレイヤーがオーナーとして記録されます。

```
[紙] [紙] [紙]
[紙] [ネザライトのクワ] [紙]
[紙] [糸] [紙]
```

### 段ボール呼び笛

紙と糸でクラフトする専用アイテム。使用すると、自分がオーナーの段ボールを全てプレイヤーの近くにテレポートさせて呼び寄せます。

```
[ ] [紙] [ ]
[紙] [糸] [紙]
[ ] [紙] [ ]
```

### 段ボール（Cardboard Box）

召喚したプレイヤーを守り、敵対Mobを狩るMob。誰彼構わず襲いかかることはありません。

- **攻撃対象**
  - 敵対Mob（ゾンビ・スケルトンなど）は無条件で攻撃します
  - オーナーが誰かを殴った・攻撃された場合、その相手を攻撃します（援護・護衛）
  - 段ボール自身が殴られた場合、殴った相手を攻撃します（反撃。ただし他の段ボールへの反撃はしません）
  - 上記に当てはまらない動物やプレイヤーは放置します
- **オーナー不可侵**: 召喚したプレイヤーは絶対に攻撃しません。殴られても反撃しません
- **同族不可侵**: 段ボール同士は互いに攻撃しません
- **飛行突撃**: 対象から離れていると、まれに重力を無視して空中を直進し対象へ突撃します。着地時はゆっくり降下（低速落下）に切り替わります
- **永続化**: ワールドを離れてもオーナー情報を保持し、遠距離でもデスポーンしません
- **オーナー死亡時に消滅**: 召喚したプレイヤーが死亡すると、その段ボールは自動的に消えます

## ビルド・実行方法

このディレクトリ自体が独立したGradleプロジェクトです。

```
cd neoforge-1.21.1
./gradlew runClient
```

初回実行時はMinecraft/NeoForgeのダウンロード・デコンパイルが走るため、時間がかかります。

MODのjarをビルドする場合:

```
./gradlew build
```

`build/libs/` にjarファイルが生成されます。
